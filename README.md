

# 雪峰 Agent — 使用指南

## 什么是雪峰 Agent？

本项目基于github上[ziqihe10\-droid / xuefeng\-agent](https://github.com/ziqihe10-droid/xuefeng-agent)开源项目开发完成，增加了相关专业数据，增强了搜索能力，蒸馏了雪峰老师8本志愿填报专著，61 节 / 1,500\+ 分钟专业视频课程等，简单来说雪峰 Agent 是一个 AI 高考志愿咨询助手，像张雪峰老师一样帮你分析志愿填报。你只需要告诉它你的情况，它就能给你个性化的建议。

### 第二步：输入姓名

首次使用时，系统会要求你输入姓名。

1. 在弹出的登录框中输入你的姓名

2. 点击「开始咨询」

3. 下次访问时会自动跳过这一步

---

### 第三步：描述你的情况

在页面底部的输入框中，直接描述你的情况。例如：

- "我是湖北考生，物理类580分，想学计算机"

- "山东620分，想学医，家里有人在医院"

- "河南文科550分，普通家庭，想考公务员"

- "广东600分，家里做电力相关工作"

**小技巧**：信息越详细，给出的建议越精准。你可以一次性说完，也可以分多次补充。

---

### 第四步：查看信息采集进度

页面左侧会显示系统已采集到的信息，包括：

- 你的省份

- 分数和位次

- 选科情况

- 专业兴趣

- 地域偏好

- 家庭资源

- 核心诉求

当这些信息都显示为 ✅ 时，说明系统已经充分了解你的情况了。

---

### 第五步：获取建议

系统会根据你的情况，给出个性化的志愿填报建议，包括：

- 能冲哪些学校

- 稳妥的选择有哪些

- 保底的方案

- 专业选择建议

- 就业前景分析

---

## 常用功能

### 快捷提问

页面提供了一些常见问题的快捷按钮，点击即可快速开始咨询：

- "湖北 580分 物理"

- "山东 620分 学医"

- "河南 550分 考公"

- "广东 600分 电力"

### 重新开始对话

如果想重新咨询，点击左侧边栏底部的「重新开始对话」按钮。

### 切换话题

你可以随时提出新的问题，比如：

- "如果不想学医了，还有什么推荐？"

- "我想了解一下XX大学怎么样？"

- "这个分数能上985吗？"

## 常见问题

### Q: 为什么提示"连接失败"？

A: 由于服务器部署在我的电脑本地，需要我的电脑打开才可以使用，服务暂时不可用时，请联系我解决

### Q: 可以咨询其他省份吗？

A: 可以，系统支持全国所有省份的高考志愿咨询。

### Q: 建议准确吗？

A: 系统基于大数据和AI分析，建议仅供参考。最终填报请结合官方信息和实际情况。

### Q: 可以多次咨询吗？

A: 可以，每次访问都是独立的会话。点击「重新开始对话」可以开启新的咨询。

### Q: 手机能用吗？

A: 可以，网站支持手机访问，界面会自动适配手机屏幕。
